HARYTECHS-CONNECT
----

📌 Project Overview

Harytechs-Connect Card is a modern web application that displays professional and personal contact information in a clean and interactive interface.

The application is built using React with Vite and Tailwind CSS and is designed to be fully responsive across all screens.

This project helps demonstrate frontend development skills including component-based architecture, responsive UI design, and modern styling techniques.

---
🚀 Features

Responsive layout for all screens

Professional Digital Business Card UI

Contact information section

Social media links (GitHub & LinkedIn)

Portfolio link

Location display

Add contact button

Clean modern interface

QR-Code scan feature

Fast development using Vite

---
🛠 Tech Stack

⚛ React

⚡ Vite

🎨 Tailwind CSS
